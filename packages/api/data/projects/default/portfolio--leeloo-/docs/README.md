# Portfolio (LeeLoo)

## Overview
A heavily animated dark-modern portfolio website featuring 3D interactive elements, neon aesthetics, and Zurto-inspired animations. Showcases 7 projects with metrics and 9 blog posts.

## Tech Stack
- **Frontend**: React 18, Three.js, Framer Motion/GSAP
- **Routing**: React Router v6
- **Styling**: CSS3 with neon effects
- **Build**: Vite

## Quick Start
```bash
npm install
npm run dev
```

## Features
- Full-screen animated hero with typing/morphing/glitch effects
- Interactive 3D manipulable elements
- 7 projects with tech breakdown & performance metrics
- 9 blog posts with cover images & excerpts
- Dark theme with neon accents
- Fully responsive design

## Project Data
- **Mock Projects**: 7 items with tech stacks, dates, metrics
- **Mock Blog Posts**: 9 items with 100-200 word excerpts
